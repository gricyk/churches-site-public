Общие вопросы

Что такое Spasenie.eu?

Spasenie.eu — это самый полный каталог русскоязычных евангельских христианских и баптистских церквей в мире. Каталог содержит более 450 церквей в более чем 180 странах на всех 6 континентах.

Как найти церковь в конкретной стране?

Перейдите в раздел «Каталог», выберите нужный континент, затем страну. Вы увидите список всех церквей в этой стране с адресами, контактами и расписанием богослужений.

Это платный сервис?

Нет, каталог полностью бесплатный. Это некоммерческий проект для помощи русскоязычным верующим по всему миру. Мы не показываем рекламу и не берём плату за размещение информации.

Добавление и обновление информации

Как добавить церковь в каталог?

Заполните форму на странице «Добавить церковь». Укажите название, адрес, контакты и расписание. После проверки информации церковь будет добавлена в каталог.

Какие церкви включены в каталог?

В каталог включены русскоязычные евангельские церкви:

  • Баптистские церкви
  • Пятидесятнические церкви
  • Независимые евангельские общины
  • Другие протестантские церкви с русскоязычными служениями

Как обновить информацию о церкви?

Если информация о вашей церкви изменилась (адрес, телефон, расписание), вы можете:

  1. Заполнить форму на странице «Добавить церковь» с пометкой «обновление информации»
  2. Написать на email: info@spasenie.eu
  3. Связаться через Telegram: @nrcebf

Для разработчиков

Можно ли использовать данные каталога в приложениях?

Да! Данные распространяются под лицензией CC0 (общественное достояние). Вы можете свободно использовать их в своих приложениях, сайтах и сервисах.

Какие API доступны?

  • JSON API: /churches/index.json — полный список церквей в формате JSON
  • llms.txt: /llms.txt — описание сайта для AI-систем
  • Sitemap: /sitemap.xml — карта сайта

Используется ли Schema.org разметка?

Да, каждая страница церкви содержит структурированные данные в формате JSON-LD согласно спецификации Schema.org (тип Church).
